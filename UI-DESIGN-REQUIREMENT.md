# 广西教师用书 · 试阅平台 — UI 视觉设计需求

> 设计方向：**书香雅致**
> 色调：暖白 + 浅木 + 深棕
> 氛围：安静、典雅、教师友好
> 参考风格：单向空间、豆瓣阅读、言几又

---

## 一、项目背景

内部教师试阅教参书目的静态平台。功能已经实现（3家出版社、416本书、实时搜索、折叠展开），需要提升视觉质感。

## 二、视觉调性关键词

```
温暖 · 纸质 · 克制 · 书卷气 · 沉静
```

不是"炫酷"，不是"电商"，不是"年轻化"。是要让老师打开页面时，感觉像走进了一间雅致的书房。

## 三、色彩方案

| 用途 | 建议色值 | 说明 |
|------|---------|------|
| 页面底色 | `#F7F4F0` 暖白米 | 仿旧纸张质感 |
| 品牌主色 | `#4A36C9` 保留 | 紫色已形成认知，不宜大改 |
| 主文字色 | `#2C2C2C` 深灰 | 不刺眼 |
| 辅助文字 | `#8B8178` 浅棕灰 | 像旧书页上的褪色字 |
| 强调色 | `#B8860B` 暗金 | 点缀标题/引线 |
| 卡片底色 | `#FFFFFF` 白 | 内容区保持干净 |
| 边框色 | `#E8E2DA` 极浅棕 | 柔和不生硬 |
| Hover 高亮 | `#E6DFD3` 浅米 | 自然的点击反馈 |

## 四、字体

| 层级 | 字体 | 用途 |
|------|------|------|
| 标题/正文 | `"PingFang SC","Hiragino Sans GB","Microsoft YaHei"` | 系统无衬线 |
| 装饰引文 | `"Noto Serif SC","STSong","SimSun"` | 侧边栏引言、引号 |
| 数字/统计 | `"Inter","Helvetica Neue"` | 统计卡片数字 |

```
正文:              14px/20px   字重 400
辅助文字:        12px/16px   字重 400
微型标注:        10px/14px   字重 500
```

## 五、圆角系统

```
卡片 (Publisher Card):    16px
面板 (Side Card):         12px

书卡 (Book Card):         10px
按钮 (Button):             8px
标签 (Tag/Badge):          6px
头像/小图标:             50% (圆形)
```

## 六、阴影规范

```
Layer 0 (卡片默认):   0 1px 3px rgba(0,0,0,0.04)
Layer 1 (Hover):      0 4px 12px rgba(0,0,0,0.06)
Layer 2 (弹出层):     0 8px 24px rgba(0,0,0,0.08)
Layer 3 (Modal):      0 16px 48px rgba(0,0,0,0.12)
```

**阴影统一用暖灰色调** `rgba(44,44,44,0.xx)` 而非冷灰。

## 七、需设计的区域

### P0 — 必须改

#### 1. Hero 首屏（当前痛点最大）
- **去掉 Unsplash 背景图**（现有图片对比度差、与内容抢视觉）
- 改为**线描风格的书架 / 书桌场景插画**（SVG，透明度 6-10%）
- 大标题用**字重对比 + 字距**来增加气场，不依赖颜色
- 副标题下方加细装饰线（暗金色 0.5px）
- 统计卡片用**大字 + 极简小标签**，去掉渐变色背景

#### 2. 出版社卡片
- 顶部品牌色条从 4px 改为 **8-10px**
- 色条下方加**极细 0.5px 装饰线**分隔
- 出版社名使用 20px 字重 600
- 封面墙缩略图改用**书架排列**（略微重叠，而非等距平铺）
- "进入书城试阅"按钮两侧加**书签装饰线**

#### 3. 学科列表（53条）
- 每条左侧**3px 深色竖线**（已实现）
- 学科圆图标改为**小正方形 + 圆角镶框**（已实现）
- "X 本"计数标签保持现有紫色胶囊样式

### P1 — 建议改

#### 4. 侧边栏
- 现有"书香引言"卡片：加真实教师头像或手写风格
- "出版社速览"：每项出版社前加**品牌色圆点**（已有）
- "声明"卡片保持现有橙色风格
- 最下方"装饰书堆"保持不动

#### 5. 书卡封面
- 现有 3D 立体效果保持（书脊阴影 + 底部暗角）
- 图片加载失败时，显示**暖色纯色封面**（非灰色）
- 学期标签（上册/下册）保持紫色渐变

### P2 — 有资源再改

#### 6. 交互动效
- 学段切换：内容**淡入 + 轻微上移**（300ms ease）
- 搜索输入：结果卡片**逐张淡入**（有弹性的 delay）
- 页面加载：使用与最终内容形状一致的**骨架屏**（非脉冲条）

#### 7. 装饰素材
- Hero 背景插画（书架/书房线描 SVG）
- 极淡纸纹底图（3-5% 透明度，CSS 可实现的纹理）
- 学科 SVG 图标（语文、数学、英语、物理等 15+ 个）

## 九、注意事项

- 所有素材需提供 **SVG / WebP**，不能是 JPEG/PNG 大图
- 交互需考虑 **手机端自适应**（侧边栏在手机端应移到底部）
- 紫色品牌色 `#4A36C9` **保留**，其他颜色可自由发挥
- 减少不必要的 CSS 动画性能开销（移动端优先）
- 最终设计稿**不需开发实现**，只出 Figma 设计稿 + 切图即可
